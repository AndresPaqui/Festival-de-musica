# Festival de Música - Proyecto Web Interactivo

Esta web simula un festival de música, diseñado para promocionar bandas, horarios y una galería de fotos. El enfoque principal fue la integración de **JavaScript** para añadir dinamismo y una mejor experiencia de usuario.

## 🚀 Tecnologías Utilizadas

El proyecto utiliza un stack de desarrollo web moderno enfocado en la interactividad:

* **HTML5**: Estructura semántica para el contenido del festival.
* **CSS3 / SASS**: Uso de preprocesadores para estilos más organizados, variables y mixins.
* **JavaScript (Vanilla)**: Implementación de lógica para la galería de imágenes, scroll suave y efectos visuales.
* **Gulp**: Automatización de tareas como la minificación de código y procesamiento de imágenes.

## 🛠️ Características Principales

1. **Galería Interactiva**: JavaScript encargado de generar una galería de imágenes dinámica con efecto de zoom (Modal).
2. **Scroll Suave**: Navegación fluida entre las diferentes secciones del sitio (Lineup, Boletos, Galería).
3. **Optimización de Imágenes**: Versiones en formatos ligeros para mejorar el tiempo de carga.
4. **Diseño Responsivo**: Totalmente adaptado para móviles, asegurando que el lineup sea legible en cualquier dispositivo.

## 💻 Visualización Local

Al incluir tareas automatizadas, puedes visualizarlo de dos formas:

1. **Clonar el repositorio:**
```bash
git clone https://github.com/AndresPaqui/Festival-de-musica.git
```

2. **Entrar al directorio:**
```bash
cd Festival-de-musica
```

3. **Ejecución Directa:**
Simplemente abre el archivo index.html en tu navegador para ver la funcionalidad de JavaScript en acción.

## 📂 Estructura del Proyecto

* **`index.html:`** Estructura principal del sitio

* **`src/js/:`** Archivos JavaScript encargados de la galería y el desplazamiento.

* **`src/scss/:`** Código fuente de estilos utilizando SASS.

* **`build/:`** Versión compilada y optimizada de los estilos y scripts.

* **`img/:`** Galería de fotos de las bandas y del evento.
